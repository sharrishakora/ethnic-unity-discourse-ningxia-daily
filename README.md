# Replication Data: Ethnic Unity Discourse in Ningxia Daily (2019-2023)

## Study Information

**Title:** The Political Function of 'Ethnic Unity' Discourse in Ningxia Daily: A Corpus-Based Study of Muslim-Related Reports, 2019-2023

**Abstract:** This study applies corpus-assisted discourse analysis with NVivo to 761 Muslim-related articles from Ningxia Daily (2019-2023). Findings reveal that economic themes dominated coverage (37.7%) while religious content remained marginal (5.6%), with government sources comprising 55.6% of citations.

## Analytical Methods

- **Software:** NVivo 12 for qualitative coding and content analysis
- **Statistical Tests:** Chi-square tests for distribution analysis
- **Reliability:** Krippendorff's alpha for inter-coder agreement (20% sample, n=152)

## Data Collection

- **Source:** Ningxia Daily digital archive
- **Period:** January 2019 to December 2023
- **Sample:** 761 Muslim-related articles
- **Search Terms:** Muslim, Hui, halal, ethnic unity, community consciousness
